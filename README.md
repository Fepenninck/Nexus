# NexusBank 

Sistema de autenticação segura desenvolvido em React + Supabase para o Projeto Integrador de Políticas de Segurança da Informação.

O projeto implementa autenticação multifator (2FA), recuperação segura de senha, proteção de credenciais e conformidade com a LGPD.

---

## Funcionalidades
- Login e cadastro de usuários
- Autenticação em dois fatores (2FA)
- QR Code para Google Authenticator
- Recuperação segura de senha
- Proteção com JWT
- Criptografia e segurança de dados
- Controle de acesso com RLS
- Conformidade com a LGPD
